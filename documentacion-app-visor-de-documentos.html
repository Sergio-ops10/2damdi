Documentacion app Visor de Documentos

Muetro el proceso que he tenido que hacer para conseguir esta aplicacion, hecha con gemini y con los siguientes prompts:

 

Actua como un experto en programador de aplicaciones web en html, css, y javascript para Crear una aplicación que podamos subirle un archivo de tipo:

Texto

PDF

Markdown

JSON

...

 

Quiero que se puedan añadir mas de un  archivo y hazlo para que pueda subir cualquier tipo de documento o archivo ej Zip, todo tipo de foto, pdf, csv, txt, en fin todo tipo de archivos

 

Añadele un boton de ajustes que habra una pestaña para configurar colores de la pagina

 

Pregunta a gemini: Q mas añadirias?

Respuesta: 

  • Barra de Búsqueda y Filtros en la Lista:

    • Por qué: Si el usuario sube 50 fotos o documentos, encontrar uno específico en la lista lateral se vuelve tedioso.

    • Solución: Un pequeño input de texto encima de la lista de archivos para filtrar en tiempo real por nombre.

  • Resaltado de Sintaxis (Syntax Highlighting):

    • Por qué: Actualmente, los archivos de código (.js, .py, .html) se ven como texto plano en negro. Es difícil de leer.

    • Solución: Integrar la librería Prism.js. Esto hará que el código se vea con colores (como en VS Code), detectando el lenguaje automáticamente.

  • Soporte de Audio:

    • Por qué: Ya soportas vídeo e imágenes. Faltaría el audio (MP3, WAV) para completar la suite multimedia.

    • Solución: Un reproductor de audio minimalista con una onda de sonido visual o controles simples.

  • Modo Edición (Editor de Texto):

    • Por qué: Ahora solo puedes "ver". Sería increíble poder hacer cambios rápidos en un JSON o corregir una errata en un Markdown.

    • Solución: Un botón "Editar" que transforme el visor en un <textarea> y permita guardar los cambios en memoria antes de descargar el archivo modificado.

  • Información de Metadatos (Detalles):

    • Por qué: A veces necesitas saber más que el nombre y el peso.

    • Solución: Un botón de "Info" que muestre detalles técnicos: dimensiones de la imagen (px), duración del vídeo, última fecha de modificación, etc.

 

Añade esas 5 actualizaciones

Añade esto: Pestañas: Permitir abrir múltiples archivos a la vez y navegar entre ellos mediante pestañas, en lugar de cerrar uno para abrir otro.
